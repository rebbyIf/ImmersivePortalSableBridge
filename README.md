
Immersive Portals Sable Bridge
=======
A mod for (a basic level of) compatibility between Immersive Portals & Sable. Special thanks to [Bunting chj](https://modrinth.com/user/Bunting_chj) for (most) of the code. Buy him a coffee: https://buymeacoffee.com/chj990707

### Known bugs & issues:
- Sable physics objects can have visual artifacts behind portals
- Worlds can rarely be stuck on the saving screen when exiting when 2 portals and a sable physics object is on the same chunk. _(The world still saves)_
- **Sable physics objects cannot go through portals**

Please report any issues on the [github](https://github.com/rebbyIf/ImmersivePortalSableBridge/issues)
